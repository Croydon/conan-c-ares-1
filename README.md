# conan-c-ares

## For Users: Use this package

### Basic setup

    $ conan install c-ares/1.14.0@conan/stable

### Project setup

If you handle multiple dependencies in your project is better to add a *conanfile.txt*

    [requires]
    c-ares/1.14.0@conan/stable

    [generators]
    txt
    cmake
